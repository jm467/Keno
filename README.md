# Keno Overview
Keno Game Application Using JavaFX
1. The project uses JavaFX to create a GUI that allows the user to play the popular casino game Keno.
2. The project uses custom classes to keep track of the game logic which is then integrated with the Graphical User Interface and displaye to the user.
3. The project also features some animations and the option to change the look of the program.

# Rules
1. Once you start the game you must first select how many spots you would like to play (1,4, 8, or 10).
2. You must then pick that many numbers in the bet card or select to randomize your selections.
3. You must then pick how many drawings you would like to play your bet card choices for (1-4).
4. Then you will go through your drawings one at a time and your winnings will be reported after each drawing.
5. Once your drawings are complete you may either quit the app or fill out another bet card.
